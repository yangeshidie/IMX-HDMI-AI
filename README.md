# IMX-HDMI-AI
基于RK3576构建嵌入式视觉系统，打通IMX415（V4L2）→DRM/KMS→HDMI显示链路；引入DMA-BUF 实现零拷贝数据传输，将端到端延迟控制在120ms内。 硬件加速与流媒体处理：基于MPP实现H.264硬件编码，满足实时访问需求。采集/编码/AI解耦的多线程架构，实现实时推理与事件触发机制，支持关键视频片段自动保存与标记。
